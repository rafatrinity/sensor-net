Descrição detalhada

Gerencia a leitura de sensores (DHT, Solo), cálculo de VPD, e armazena os resultados em cache. Atualiza um cache interno thread-safe e executa uma tarefa dedicada para leituras periódicas. Utiliza RAII para recursos.
