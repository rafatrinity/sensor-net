Descrição detalhada

Manages the ESPAsyncWebServer, handling HTTP requests and Server-Sent Events (SSE).

This class is responsible for initializing the web server, setting up routes (though route setup is typically done in the .cpp file), and sending real-time updates to connected clients via SSE.
