Descrição detalhada

Manages the control of actuators like lights and humidifiers.

This class is responsible for initializing and controlling actuators based on target values (e.g., from TargetDataManager) and current conditions (e.g., time from TimeService, sensor readings from SensorManager). It uses FreeRTOS tasks for continuous control loops.
