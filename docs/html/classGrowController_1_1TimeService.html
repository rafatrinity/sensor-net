◆ getCurrentTime()

bool GrowController::TimeService::getCurrentTime ( struct tm &  timeinfo) const

Obtém a hora local atual sincronizada via NTP. Tenta obter a hora atual do sistema, que deve estar sendo sincronizada pelo cliente SNTP do ESP32 (configurado em initialize()).

Obtém a hora local atual sincronizada via NTP.

Parâmetros
timeinfoReferência a uma struct tm que será preenchida com a hora atual se a função for bem-sucedida.
Retorna
true Se a hora local foi obtida com sucesso (implica que o NTP está sincronizado).
false Se não foi possível obter a hora local (serviço não inicializado ou NTP não sincronizado ainda).

◆ initialize()

bool GrowController::TimeService::initialize ( const TimeConfig &  config)

Inicializa o serviço de tempo, configurando o NTP. Deve ser chamado após a conexão WiFi ser estabelecida.

Inicializa o serviço de tempo, configurando o NTP.

Parâmetros
configReferência à estrutura TimeConfig contendo o offset UTC e o servidor NTP.
Retorna
true Se a configuração inicial do NTP foi chamada e (opcionalmente) a sincronização inicial foi bem-sucedida.
false Se houve um problema na configuração (ex: config inválida).

Verifica se o serviço de tempo foi inicializado (tentativa de configuração NTP ocorreu). Não garante que o NTP esteja atualmente sincronizado.

Verifica se o serviço de tempo foi inicializado com sucesso.

Retorna

false Se initialize() ainda não foi chamado ou falhou na configuração.
