\doxysubsection{Descrição detalhada}
Manages the control of actuators like lights and humidifiers. 

This class is responsible for initializing and controlling actuators based on target values (e.\+g., from \mbox{\hyperlink{classGrowController_1_1TargetDataManager}{Target\+Data\+Manager}}) and current conditions (e.\+g., time from \mbox{\hyperlink{classGrowController_1_1TimeService}{Time\+Service}}, sensor readings from \mbox{\hyperlink{classGrowController_1_1SensorManager}{Sensor\+Manager}}). It uses Free\+R\+T\+OS tasks for continuous control loops.
