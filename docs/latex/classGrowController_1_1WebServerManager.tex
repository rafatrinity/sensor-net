\doxysubsection{Descrição detalhada}
Manages the E\+S\+P\+Async\+Web\+Server, handling H\+T\+TP requests and Server-\/\+Sent Events (S\+SE). 

This class is responsible for initializing the web server, setting up routes (though route setup is typically done in the .cpp file), and sending real-\/time updates to connected clients via S\+SE.
